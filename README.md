# Global Administrator Permissions

A comprehensive guide explaining what Global Administrators can and cannot access in Microsoft Teams and Microsoft 365.

## About This Site

This site is designed to help non-technical people understand the limitations and capabilities of Global Administrator permissions in Microsoft 365. It specifically addresses misconceptions about file access through Teams and provides evidence of audit trails.

## Key Topics Covered

- What Global Admins CANNOT do through Teams interface
- What Global Admins CAN do with proper administrative tools
- Demonstration of Teams Files tab search limitations
- Evidence and audit trails in Microsoft 365
- Access scenarios and their audit implications
- Technical glossary for non-technical readers

## Live Site

Visit the live site: [https://wellis321.github.io/global-admin-permissions](https://wellis321.github.io/global-admin-permissions)

## Purpose

This site was created to address accusations that Global Administrators can bypass normal permission structures to access user files through Teams. It demonstrates that such access is technically impossible through standard interfaces and that any legitimate administrative access is fully logged and auditable.
